- Atividade: Desenvolver o Modelo JSON em Java, que tera sua saída gerada através do console;
	- Metodologia da atividade: 

		1 Imagem em anexo com o contexto da atividade;
		2 Levantar as informações e definições necessarias para gerar JSON
		3 Descreve o modelo de Json com suas caracterisitcas;
		4 Modelo em txt disponível na raiz do projeto com o nome modelo-de-json;
		5 Gerar o modelo de json a partir do console do sistema;

- Objetivo: JSON será utilizado como modelo para alimentar a IA;

Descrição matematica do Json referido

- Dentro do conjunto context existem sub conjuntos sendo em sequencia: import, source e intents
- Dentro de intents exitem subconjuntos sendo na sequencia: intent, intent-generated, code, entities e values.
- Dentro do subconjunto intents a estrutura de subconjuntos pode se repetir em um laço de repetição.
- No sub conjunto code exite um subconjunto dentro dele chamado selenium que tambem possui um subconjunto chamado java.
- No subconjunto entities exitem dois subconjuntos chamados target e related
